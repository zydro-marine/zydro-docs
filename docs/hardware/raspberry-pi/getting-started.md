# Getting Started with Raspberry Pi

This guide will help you get started with your Raspberry Pi.

## What You'll Need

- Raspberry Pi board
- MicroSD card (32GB or larger recommended)
- Power supply
- HDMI cable and display (optional)
- Keyboard and mouse (optional)

## First Steps

1. Download Raspberry Pi OS
2. Flash it to your microSD card
3. Insert the card into your Raspberry Pi
4. Power on and follow the setup wizard
